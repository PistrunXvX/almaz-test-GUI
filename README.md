# Тестовое задание для НПО Алмаз

## Задание 1 | GUI
Написать программу с графическим интерфейсом на GDScript, с помощью которой можно открыть предложенный csv файл и вывести его содержимое в виде таблицы, помещенной внутрь окна.  Элементы таблицы должны динамически менять размер в зависимости от размеров окна (растягиваться, сжиматься). Должен появляться вертикальный скролл в случае, если элементы таблицы не помещаются в нее по оси Y. Шапка таблицы (тип самолета, пасcажировместимость, крейсерская скорость) должна быть всегда видна. Включать подсветку элемента зеленым цветом, когда курсор мыши находится на нем.

## Как работает
 - Кнопка загрузить csv файл - загружает спарсенные данные в таблицу

## Задание 2 | 3D
В Godot Engine 4 (предпочтительно) или Unity реализовать 3D-визуализацию движения тележки, составленной из примитивных форм (корпус - параллелерипед, колеса - цилиндры) по плоской горизонтальной поверхности. Траектория движения - синусоида, тангенциальное ускорение в процессе движения отсутствует. Колеса должны вращаться в соответствии со скоростью движения тележки, передние колеса должны также поворачиваться в соответствии с параметрами ее поворота. Через UI до начала симуляции должны задаваться значения параметров: скорость движения тележки, амплитуда и период синусоиды-траектории. Камера должна управляться с клавиатуры по позиции и направлению (WSAD, мышь по двум осям).

## Как работает
- Запускать из сцены **cargo_wheel.tscn**
- Тележка реализованна через ноду **VehicleBody3D**. Присутствует физика тел
- Путь синусоиды генерируется через **Path3D**
