# Тестовое задание для НПО Аламаз

## Задание
Написать программу с графическим интерфейсом на GDScript, с помощью которой можно открыть предложенный csv файл и вывести его содержимое в виде таблицы, помещенной внутрь окна.  Элементы таблицы должны динамически менять размер в зависимости от размеров окна (растягиваться, сжиматься). Должен появляться вертикальный скролл в случае, если элементы таблицы не помещаются в нее по оси Y. Шапка таблицы (тип самолета, пасcажировместимость, крейсерская скорость) должна быть всегда видна. Включать подсветку элемента зеленым цветом, когда курсор мыши находится на нем.

## Как работает
 - Кнопка загрузить csv файл - загружает спарсенные данные в таблицу
